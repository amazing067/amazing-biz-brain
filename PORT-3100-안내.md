# 치매검사(amazing-biz-brain) 개발 서버 포트 3100 안내

## 원인 정리

- **3000으로 뜨던 이유:** 터미널 **현재 폴더**가 `amazing-biz-blog`일 때 `npm run dev`를 실행했기 때문입니다.
- `npm run dev`는 **현재 폴더의 package.json**을 사용합니다.
  - **amazing-biz-blog** → `"dev": "next dev"` → 포트 **3000** (기본값)
  - **amazing-biz-brain** → `"dev": "cross-env PORT=3100 next dev -p 3100 -H 0.0.0.0"` → 포트 **3100**

## 올바른 실행 방법

1. 터미널에서 **반드시 이 프로젝트 폴더로 이동**한 뒤 실행하세요.
   ```powershell
   cd c:\Users\ok\amazing-biz-brain
   npm run dev
   ```
2. 실행되면 다음처럼 **3100**으로 표시됩니다.
   ```
   ▲ Next.js 16.1.3 (Turbopack)
   - Local:         http://localhost:3100
   - Network:       http://0.0.0.0:3100
   ```
3. 브라우저에서 **http://localhost:3100** 로 접속하세요.

## 확인한 내용

- **amazing-biz-brain** 폴더에서 `npm run dev` 실행 시 → **localhost:3100** 로 정상 기동 확인됨.
- Cursor에서 터미널을 열면 이전에 **blog** 폴더에서 켜 둔 상태일 수 있으므로, 치매검사 서버를 띄울 때는 **먼저 `cd c:\Users\ok\amazing-biz-brain`** 한 다음 `npm run dev`를 실행하면 됩니다.
